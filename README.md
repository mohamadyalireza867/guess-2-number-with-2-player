# Two-Player Guessing Game

A simple Python console game where two players compete to guess a secret number generated by a "Dealer."

## How to Play
1. The Dealer generates a random number between 0 and 10.
2. Player 1 and Player 2 take turns guessing the number.
3. The program provides hints (Greater/Smaller).
4. The player who guesses the number in the fewest attempts wins!

## Requirements
- Python 3.x

## How to Run
```bash
python guessing_game.py
